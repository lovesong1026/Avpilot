# Avpilot v0.1 功能契约

## 目标

v0.1 打通从资料入库到可追溯问答、长期记忆和可视化的完整闭环。实现参考 Comet 的外部能力，但采用 Avpilot 自己的数据模型、接口契约、提示词和测试集。

## 功能与验收

### 内容入库与 RAG

- 支持 PDF、DOCX、Markdown、TXT、HTML、网页 URL 和图片。
- 文档解析保留页码、标题层级、来源 URL 和字符范围。
- 父块用于生成上下文，子块用于精确召回；两者使用稳定 ID 关联。
- Elasticsearch 同时建立 IK 中文全文字段和 1024 维向量字段。
- BM25 与向量召回使用 RRF 融合；Rerank 可关闭，失败时自动回退融合排序。
- 每条回答引用包含来源类型、标题、原对象 ID、定位信息、原文摘录和得分。

### 图片多模态

- 原图保存在文件存储，PostgreSQL 保存元数据。
- 百炼视觉模型输出描述、OCR、物体列表和场景。
- 描述与 OCR 合并后向量化并进入 Elasticsearch。
- 图片失败不影响同批其他资料，支持单项重试。

### 自动标签

- 分类器优先从用户已有标签中选择。
- 只有无合适标签时才提出新标签；新标签使用规范化名称去重。
- 文档和图片都支持人工增删标签，人工结果优先于自动结果。

### 记忆

- 支持主动记住和对话结束后异步提取。
- 图谱保持 `Source → Chunk → Statement → Entity` 四层来源链。
- Event 单独保存发生时间、结束时间和地点，并连接相关实体。
- 第一层使用规范化名称和向量候选去重；第二层使用 LLM 判断候选是否同一实体。
- 社区聚类是可重复执行的派生结果，不改变原始陈述。

### Agent 问答

- 内建 knowledge、memory、web_search 三个工具。
- 支持 function calling；模型不支持工具调用时才使用受限 ReAct。
- 每次工具循环限制轮数、超时和输出长度。
- SSE 事件固定为 `start/tool_start/tool_result/delta/citation/usage/done/error`。
- 支持文字加图片提问，并在多模态模型不可用时返回明确错误。

### 搜索与导航

- 全局搜索分别返回文档、图片和记忆三组结果。
- 每组配置最低语义分数，低于阈值的结果不展示。
- 支持收藏文档、图片、消息和记忆；收藏保留展示快照。
- 每日回顾汇总当天新增资料、对话和记忆。

### 可视化

- AntV X6 展示实体、陈述、来源和关系，可按类型筛选。
- 时间线按事件时间排序并可回到来源。
- ECharts 展示资料量、处理成功率、检索次数、记忆增长和模型用量。

## 交付顺序

1. 身份、模型适配、数据表和 ES/Neo4j schema。
2. 文档纵向闭环：上传、解析、分块、Embedding、检索、引用。
3. 图片、网页和自动标签。
4. 对话与 Agent SSE。
5. 记忆图谱、去重、聚类和回顾。
6. 全局搜索、收藏与三类可视化。
7. 离线评测、安全测试和端到端回归。
